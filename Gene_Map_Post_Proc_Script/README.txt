Genetic mapping experiment post-processing scripts

Zihao Li, Dec 2022

FOLDER DESCRIPTIONS:

This folder contains genetic mapping data acquired by WormPicker (archived as text files) and MATLAB scripts for processing these data.
MATLAB scripts are able to generate bar charts for depicting linkage patterns (Supplmentary Fig. 13a-b), and to calculate the linkage indices (Supplmentary Fig. 13d).


HOW TO USE THE FILE:

1.	Double click the "Main.m" and the main processing script can be opened in MATLAB
2.	Run the "Main.m" by hitting "Run" buttom
3.	The plots can be visualized
